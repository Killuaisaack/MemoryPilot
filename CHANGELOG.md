# Changelog

## 3.4.2

- 修复 TauriTavern 环境下面板底部适配问题，改善最后一条记忆的编辑、删除按钮被遮挡的情况。
- 为记忆列表删除操作增加撤回入口，单条删除和批量删除后都可以恢复刚删除的记忆。
- 增加存储日志查看、复制与清空入口，方便用户反馈问题时提供诊断信息。
- 优化记忆存储可靠性，增加本地备份、恢复快照、损坏缓存回退和更严格的变更检测，降低记忆丢失风险。
- 增加首次使用指引，并在过滤页保留手动打开入口；明确 XB 事件导入后必须重构关键词。
- 优化记忆列表搜索体验，搜索后点击事件标题可回到完整列表并跳转到对应记忆位置。
- 增加记忆列表回顶、触底悬浮按钮，提升长列表浏览体验。
- 增加扩展设置区的版本检查入口，可对比本地版本与仓库最新版本，并提示用户前往酒馆扩展管理器更新。
